start()和run()到底有什么区别

首先需要明确的是，start()是一个native方法，该方法会启动一个线程，线程执行run()方法里的内容。与直接执行run()方法不同的是，start()会启动一个新的线程，这个线程与主线程并发执行。如果直接执行线程的run()方法就不会启动新的线程，相当于执行了一个普通方法。
